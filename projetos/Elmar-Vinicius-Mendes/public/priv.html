Política de Privacidade — Zerion

Última atualização
17/05/2026
1. Coleta de dados
A Zerion pode coletar dados fornecidos voluntariamente pelo usuário, como nome, e-mail e informações inseridas durante o cadastro.
2. Finalidade do uso
Os dados coletados são utilizados exclusivamente para simulação de funcionalidades dentro do ambiente acadêmico do sistema.
3. Armazenamento
Por se tratar de um projeto acadêmico, não há garantia de armazenamento permanente, podendo os dados ser resetados ou removidos a qualquer momento.
4. Compartilhamento
Nenhuma informação é compartilhada com terceiros ou serviços externos, uma vez que o sistema não possui integração com plataformas reais.
5. Segurança
Não há garantia de segurança absoluta dos dados, dado o caráter experimental e educacional do projeto.
6. Observação final
Este sistema é uma simulação acadêmica sem caráter comercial ou operacional real.
